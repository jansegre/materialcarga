Sobre


Plataforma

Esta plataforma foi o resutado de um TD (Tema Dirigido) concluído ao final de 2011.

O projeto é opensource com uma licensa Apache e está disponível aqui, qualquer problema deve ser reportado aqui.

O site foi desenvolvido usando o Play! framework e está implementado em Java.


Recomendações

Este site segue os padrões propostos pela W3C para as tecnologias HTML5 e CSS3, e também utiliza JavaScript.

É recomendado usar as versões mais recentes dos grandes navegadores, atualmente:

  • Internet Explorer (9.0+)
  • Firefox (7.0.1+)
  • Google Chrome (15.0+)
  • Safari (5.1.1+)
  • Opera (11.52+)
